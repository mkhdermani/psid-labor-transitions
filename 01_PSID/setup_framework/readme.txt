This should be a roughly self-contained set of code to download and generate PSID code to run in Stata.

The first thing to do is go to the "keyDocs" directory and open the "psid.xlsx" file. This contains a crosswalk of all variables in the PSID. First pick the subset of variable (by row) that you would like in your final analysis. You'll want to save the "Family" variables separately from the "Individual" variables. Examples of what this should look like are in "Family_Variable_Selection.xlsx" and "Individual_Variable_Selection.xlsx"

Once you do this, go up one directory and then into the "genDoFiles" directory. There is a file called "PSIDMaster.py" You'll need python3 and a couple dependencies to make this work. Here you have a couple of options: 

i) first, you can download all of the PSID data. This requires that you make a log in at the ISPCR website and enter it in the "opts" container in the PSIDMaster.py file. If you want to do this, you will need to do some extra work to get the downloader to work (it did in 2018)--run it and if you run into errors, start debugging. The code for this is in at "./src/DownloadFunctions.py". 

ii) you can create a set of stata do-files that will create a panel dataset from the variables that you selected at the beginning. It will organize the code as follows: It loops through the categories of the variables that you have selected and creates a do file that keeps each of your selected variables across all years it is included in the PSID questionnaire. Then it creates a file that will run each of these do-files (once you ultimately run the code in Stata). Next, it creates a do-file that will reshape all of your variables into a panel, where the reshape command is defined by year and a hardcoded individual ID. Finally, it renames the variables in a systematic (but potentially confusing, sorry) way, starting with the first 3 letters of its category with a unique number appended to the end (that DOES NOT vary with your subset, it is specified based on its position in the "psid.xlsx" file, so this is something of a saving grace). Last, it labels each of these variables based on the question information provided in the "psid.xlsx" file.

Set your paths appropriately in the "PSIDMaster.py" file and it will write and output all the required Stata code to generate panels after you run the file. Note that you may need to uncomment the last "if __main__ " line and subsequent lines and comment out the lines immediately preceding depending on how you're running the code.

Great! Now you're ready to create your PSID panel. Go back to the root directory and there will be a directory called "psidCode." It contains a file called "PSIDMaster.do" and two directories: "data_prep" and "analysis". All you need to do is move the code generated by the Python code into the data_prep folder and then run the PSIDMaster file after you appropriately set the paths.

I include some of my own customizations in the "*Clean.do" files. You'll need to clean the data after it is generated, but this will depend on your own needs.

Last, there is sample code to analyze the data in the "analysis" directory.

Have fun!
